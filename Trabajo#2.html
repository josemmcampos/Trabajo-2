HTML (HyperText Markup Language) Es el lenguaje de marcado que se usa para estructurar el contenido de una página web. Define los elementos como títulos, párrafos, imágenes, enlaces y más..

CSS (Cascading Style Sheets) Es el lenguaje de estilos que se usa para darle diseño y apariencia a una página web. Permite cambiar colores, fuentes, márgenes, posiciones y hacer diseños responsivos con media queries.

JavaScript (JS) Interactividad y Dinamismo Permite agregar efectos, animaciones y hacer que la web responda a la interacción del usuario. Puede cambiar el contenido sin recargar la página.
